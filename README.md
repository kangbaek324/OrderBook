    # 소개

한국 거래소 호가창을 기반으로 한 호가창 구현 프로젝트 입니다.

# 기능

### user
* 회원가입
* 로그인
* 계좌 개설
### stock
* 주식 매수
* 주식 매도
* 주식 주문 정정
* 주식 취소
* 미체결 주문 조회
* 체결 주문 조회
* 전체 주문 조회

</br>
상세한 API 정보는 /api 에서 확인하실수 있습니다.
